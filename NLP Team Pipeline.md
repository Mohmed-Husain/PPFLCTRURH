# NLP Team Pipeline
## Goal
Build a production-ready Hierarchical Multi-Label Text Classification (HMLTC) model that can later be plugged into the Federated Learning framework.

---

## Phase 1 — Literature & Planning

### Tasks
- Study HMLTC papers
- Understand ICD hierarchy
- Study baseline architectures
- Finalize datasets
- Define evaluation metrics
- Design project structure

### Deliverables
- Literature summary
- Finalized model architecture
- Dataset selection
- Experiment plan

---

## Phase 2 — Dataset Preparation

### Tasks
- Download dataset
- Clean clinical text
- Remove invalid records
- Prepare hierarchical labels
- Train / Validation / Test split
- Build preprocessing pipeline

### Deliverables
- Clean dataset
- Label mapping
- Preprocessing scripts

---

## Phase 3 — Model Development

### Tasks
- Implement HMLTC model
- Implement tokenizer
- Configure training pipeline
- Implement loss functions
- Implement evaluation metrics

### Deliverables
- Working HMLTC implementation

---

## Phase 4 — Standalone Training

### Tasks
- Train model
- Hyperparameter tuning
- Save checkpoints
- Evaluate performance
- Export best model

### Deliverables
- Trained model
- Model weights
- Tokenizer
- Configuration files

---

## Phase 5 — Integration Support

### Tasks
- Package model
- Document input/output format
- Support FL integration
- Fix compatibility issues

### Deliverables
- Integration-ready model

---

## Phase 6 — Evaluation

### Tasks
- Evaluate federated checkpoints
- Generate metrics
- Create graphs
- Create tables
- Compare centralized vs federated

### Deliverables
- Evaluation report
- Figures
- Tables

---

## Final Deliverables

- Clean dataset
- Preprocessing pipeline
- HMLTC implementation
- Trained model
- Checkpoints
- Evaluation scripts
- Result visualizations
- Documentation
